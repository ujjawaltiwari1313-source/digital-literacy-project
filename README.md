# Digital Literacy Project

This project is developed as part of the CSE0001 course. It focuses on understanding digital literacy, use of online platforms, and safe internet practices.

---

## Tasks Completed

Task 1: Infographic  
An infographic on Digital Literacy Awareness was created using Canva.

Task 2: Professional Profile  
A LinkedIn profile was created to build professional presence and networking.

Task 3: Coding Platform  
A basic problem was solved on HackerRank to demonstrate coding platform usage.

Task 4: Email Etiquette  
A formal email was written highlighting the importance of digital literacy.

Task 5: Cybercrime Awareness  
Content was prepared explaining types of cybercrime and methods of prevention.
